# Presentación TFM — Trivial accesible

Presentación de la defensa del Trabajo Final de Máster **«Diseño y desarrollo de un modelo accesible para juegos de preguntas y respuestas: aplicación al Trivial»**.

Máster en Accesibilidad Digital para Educación, Comunicación y Diseño — Universitat de Barcelona, junio de 2026.
Autor: Jose María Ortiz Silva.

## Cómo ver la presentación

Abre `index.html` en un navegador moderno (Chrome, Edge o Firefox).

> Se recomienda abrirla directamente en el navegador y no en la vista previa de un editor, ya que el sistema de navegación depende de los atajos de teclado.

## Navegación con el teclado

| Tecla | Acción |
|-------|--------|
| Espacio · → · AvPág | Avanzar un paso |
| ← · RePág | Retroceder un paso |
| Mayús + → · Mayús + AvPág | Diapositiva siguiente |
| Mayús + ← · Mayús + RePág | Diapositiva anterior |
| Inicio · Fin | Primera / última diapositiva |
| ↓ · . | Repetir el último anuncio |
| ↑ | Anunciar la hora |
| F | Pantalla completa |

## Accesibilidad

La presentación está diseñada para ser utilizada con lector de pantalla: el contenido visual y los anuncios de voz se controlan de forma independiente mediante un sistema de revelación por pasos.

## Estructura del proyecto

- `index.html` — presentación principal
- `css/` — estilos
- `js/` — componentes del sistema de presentación
- `img/` — imágenes
- `video/` — vídeos de la prueba de concepto
